# Task Manager API

## 📌 Description
RESTful API for managing tasks using Node.js and Express.

## 🚀 Features
- Create Task
- Get Tasks
- Update Task
- Delete Task
- Filtering by completion status

## 🛠️ Tech Stack
- Node.js
- Express.js

## ▶️ How to Run
npm install  
npm start  

## 🧪 Run Tests
npm run test  

## 📮 API Endpoints

| Method | Endpoint | Description |
|--------|--------|-------------|
| POST | /tasks | Create task |
| GET | /tasks | Get all tasks |
| GET | /tasks/:id | Get task |
| PUT | /tasks/:id | Update task |
| DELETE | /tasks/:id | Delete task |
